# Motoka - Gestion d'Agences de Transport

Motoka est une application SaaS "Mobile First" conçue pour digitaliser et centraliser la gestion des agences de transport.

## Documentation
- [Fiche Technique](./FICHE%20TECHNIQUE%20—%20MOTOKA.md)
- [Documentation Backend (Intégration)](./backend.md)

## Tech Stack
- **Frontend**: Next.js (App Router), TypeScript, TailwindCSS
- **Backend**: Laravel API (Sanctum)
- **State**: Zustand, localforage (Offline-first)
